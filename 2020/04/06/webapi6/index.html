1. touch事件类型

​ 移动设备上无法使用鼠标，当手指触碰屏幕时会触发 click、mousedown、mouseup 事件。

但是，touch事件要比鼠标事件执行效率高，用户体验好。

注意：以下事件必须通过事件监听注册

事件添加：

​ 元素.onclick = function () {}

​ 元素.addEventlistener(‘click’,function (){});

  • touchstart，手指按下事件
  • touchmove，手指移动事件
  • touchend，手指松开事件

2. touch事件对象

  • 常见的属性

    1. 事件对象.touches ：位于屏幕上的所有手指的列表；
    2. 事件对象.targetTouches ：位于该元素上的所有手指的列表；
    3. 事件对象.changedTouches：被改变的手指列表。 touchstart时包含刚与触摸屏接触的触点，touchend时包含离开触摸屏的触点

    touches和targetTouches必须是在屏幕上，而changedTouches可以获取离开屏幕的手指的

  • 手指的位置

    1. 手指对象.clientX/Y 手指相对于可视区域
    2. 手指对象.pageX/Y 手指相对于文档

    注意：手指对象.clientX/Y 使用较多

Zepto

概念

Zepto是一个轻量级的 针对现代手机端浏览器的 JavaScript 库。【JS文件】

因为 Zepto 封装了移动端的手势模块，直接把很多手机端手势直接封装成了事件，我们可以直接使用。

Zepto 中文官网地址：https://www.html.cn/doc/zeptojs_api/

注意：手机端支持良好，PC端最好不要使用.

javaScript 库介绍

由其他的团队把我们常用的功能，直接封装好了，按照提供的手册学习使用即可。

所有 JavaScript 库，本质就是 JavaScript 编写的代码，所有底层的 API 都是一样通过 JS 提供的API进行封装。

但是在调用 JS 库的属性和方法的时候，因为会在原来的基础是包装过，所以要按照手册来使用。

zepto使用模块

使用 zepto 步骤

  1. 引入 zepto.js 核心库。
  2. 引入 zepto.touch.js 移动端手势模块。
  3. 新建 <script> 标签写业务代码。
